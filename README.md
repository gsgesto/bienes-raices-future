## Bienes Raices Future

Programa de consola para Proyecto Integrador I.


### Para comenzar:

- Crear un entorno virtual y activarlo
  
  ```python3 -m venv myenv```
  
  ```source myenv/bin/activate```

- Instalar tabulate

  ```pip install tabulate```

- Correr el archivo principal

  ```python3 main.py```
